📋 Langkah-langkah Penggunaan

  1. Guest Mode - Klik "GUEST" untuk masuk tanpa daftar
  2. Buat Akun - Daftar dengan username & password
  3. Login - Masuk dengan akun yang sudah terdaftar
  4. Lihat File ZIP - Klik menu "File ZIP"
  5. Download File - Klik tombol Download
  6. Rating & Komentar - Buka menu "Rating Map"
  7. Ganti Tema - Klik toggle switch di pojok kanan
  8. Inbox - Cek pesan dari admin
  9. Lapor - Kirim saran/laporan ke admin

👤 Perbedaan Role

  • 👻 Guest - Hanya lihat file
  • 👤 User - Bisa download, rating, komentar
  • 👑 Admin - Akses penuh: upload, hapus, kelola akun, balas pesan
